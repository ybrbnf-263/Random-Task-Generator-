# Random-Task-Generator-
## Описание программы
"Random Task Generator" – это удобное приложение с графическим интерфейсом (GUI), предназначенное для генерации случайных задач. Приложение позволяет вести учет сгенерированных задач, фильтровать их по типу (например, "Учёба", "Спорт", "Работа") и сохранять историю в файле JSON. Это поможет вам не забывать о делах и поддерживать продуктивность.

## Примеры использования
Генерация задачи: Нажмите кнопку "Сгенерировать задачу", чтобы получить новую случайную задачу. Задача будет отображена во всплывающем окне и добавлена в историю.
Фильтрация: В выпадающем списке "Фильтр по типу" выберите нужную категорию задач (например, "Учёба"). Будут отображаться только задачи этого типа. Выберите "Все", чтобы увидеть полный список доступных задач.
История задач: Список всех сгенерированных задач отображается в нижней части окна.
